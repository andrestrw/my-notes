# Application Deployment Notes

## Lenguaje Config Files

On heruku the lenguaje buildpack will be detected based on the lenguaje config file.

| Lenguaje | Config File |
|----------|--------------|
| JavaScript | package.json |
| Ruby | Genfile |
| Python | requirement.txt |
| Rust | Cargo.toml |
